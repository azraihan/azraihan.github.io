## Overview

**CodeSprout** is an AI-powered coding education platform designed to make programming accessible and engaging for children through gamified, block-based learning. Built for Therap JavaFest 2024, where it ranked **Top 10 among 1300+ participants**.

The platform combines computer vision, generative AI, and a conversational ML chatbot to create a personalized learning environment that adapts to each child's pace and progress.

---

## Features

**Block-based programming interface** — Visual, drag-and-drop programming blocks that introduce computational thinking without syntax barriers.

**AI-powered chatbot** — ML-driven conversational agent providing personalized activity suggestions and adaptive content recommendations based on individual learning history.

**Computer vision integration** — Real-time drawing-to-image generation using CV models, enabling children to see their creative sketches come to life as generated images — reinforcing engagement.

**Automated project generation** — System generates personalized coding challenges calibrated to each learner's current level and progress trajectory.

**Parental progress dashboard** — Analytics and progress tracking for parents and educators to monitor learning outcomes.

## Architecture

Built on a **Java/Spring Boot microservices backend** with a **React frontend**, integrating multiple AI components:

- ML recommendation engine for adaptive content delivery
- Vector database for semantic similarity in content retrieval
- Computer vision pipeline for sketch-to-image generation
- Generative AI for dynamic project prompt creation

**Stack:** Java · Spring Boot · React · Python · PyTorch · Computer Vision · Vector Database · Gen AI
